# Simple HTTPS Proxy - Written in Go
Developed with go
```
go version go1.15.3 linux/amd64
```

Simple HTTPS proxy, using CONNECT pattern

And with Goroutines =)

## USAGE

Simple run the proxy
```
with binary
./go-proxy
OR from source
go run cmd/go-proxy/main.go
```
